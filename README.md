# CampusGuide 2026 - Events & Clubs

**Author:** Adnan  
**Course:** Web Development Assignment  
**Year:** 2026  

---

## Project Overview

CampusGuide 2026 is a **multi-page university website** designed to help students discover events, clubs, and activities on campus.  
The site includes:  

- Event highlights and featured activities  
- A schedule page with tabular data  
- A contact form for inquiries and registration  
- Accessible and semantic HTML structure  
- Responsive design for mobile and desktop  

---

##  Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page with welcome message, featured events, and latest updates |
| `content.html` | Detailed list of campus events with descriptions |
| `data.html` | Schedule page with a real table of events |
| `contact.html` | Contact page with accessible form including name, email, faculty, year, subject, message, and subscribe checkbox |

---

##  Features & Implementation

- **Semantic HTML**: `header`, `nav`, `main`, `section`, `article`, `footer`  
- **Navigation**: Same menu on all pages, working links between pages  
- **Forms**: Accessible labels, fieldsets, legends, required fields, keyboard-friendly  
- **Tables**: Proper `<th>` elements, caption, zebra striping for readability  
- **CSS**: Single external stylesheet with typography, spacing, navigation styling, table styling, form styling, responsive design  
- **Accessibility**: Skip link, keyboard navigation, readable color contrast
